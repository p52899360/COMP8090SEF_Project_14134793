# OOP-Based Library Management System

Brief Introduction
A simple library management system written with Python OOP. It can add books, check all books, change library name, verify book IDs, and sort books by stock. This is a preliminary version with basic functions only.

File Structure
- book.py: Define Book parent class and Ebook subclass
- library.py: Implement library management functions (add/check/sort books)
- data_structure_heap.py: Implement MaxHeap data structure
- heap_sort.py: Implement heap sort algorithm with custom key support
- main.py: Main program entry, run this file to start the system

How to Run
No third-party libraries needed. Just run the main.py file directly in Python environment:
```bash
python main.py
